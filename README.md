# Gra Puzzle w C++

## Opis Projektu
Projekt gry "Puzzle" został zrealizowany jako część kursu na Wydziale Elektroniki i Informatyki, skupiając się na programowaniu komputerów i urządzeń. Gra pozwala użytkownikom na składanie obrazka z małych, losowo wymieszanych fragmentów. Celem jest ułożenie wszystkich kawałków obrazka we właściwe miejsce. Gra została zaimplementowana w języku C++ z wykorzystaniem .NET Framework 3.5 i obejmuje graficzny interfejs użytkownika oraz ścieżki dźwiękowe, które uatrakcyjniają rozgrywkę.

## Funkcjonalności
- **Interaktywne układanie puzzli:** Gracze układają fragmenty obrazka w celu odtworzenia pierwotnego obrazu.
- **Zliczanie kroków:** Gra monitoruje liczbę ruchów wykonanych przez gracza.
- **Ścieżki dźwiękowe:** Gra zawiera ścieżki dźwiękowe, które uatrakcyjniają rozgrywkę oraz zwiększają immersję.
- **Interfejs użytkownika:** Graficzny interfejs użytkownika ułatwia nawigację i interakcję z grą.
